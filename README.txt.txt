# GIS Project Folder Structure Template

This template organizes GIS project files efficiently. Below is the structure:

- **Raw_Data/**: Contains all raw, unprocessed data.
  - `Vector/`: For vector data like shapefiles or feature classes.
  - `Raster/`: For raster data like satellite imagery or DEMs.
  - `Tabular/`: For tabular data like CSV or Excel files.

- **Processed_Data/**: Stores all processed datasets.
  - `Interim/`: For intermediate files during data processing.
  - `Final/`: For finalized data products ready for use.

- **Documentation/**: Stores metadata and project documentation.
- **Maps/**: Contains map project files and exports.
- **Project_Files/**: For any additional project-specific files.
